Quiz 3

\[ y_i = \beta_0 + \beta_1x_{1i} + \beta_2x_{2i} + u_i \]

  1. In the multiple regression model above, the zero conditional mean (“exogeneity”) assumption for unbiasedness means:

    1. Either \(\text{E}(u_i|x_{1i}) = 0\) OR \(\text{E}(u_i|x_{2i}) = 0\)
    2. \(\text{E}(u_i|x_{1i},x_{2i}) = 0\)
    3. \(\text{E}(\hat{\beta}_0) = 0\)
    4. Either \(\text{E}(\hat{\beta}_1) = 0\) OR \(\text{E}(\hat{\beta}_2) = 0\)

  2. In the model above, \(\beta_1\) is the effect of \(x_1\):

    1. holding \(x_2\) fixed, constant
    2. holding \(x_2\) at zero, but not at other values
    3. only if \(x_1\) and \(x_2\) are uncorrelated
    4. when \(y\) is at its mean

  3. For the model above, the residual for unit \(i\) is:

    1. \(\hat{\beta}_1x_{1i} + \hat{\beta}_2x_{2i}\)
    2. \(\hat{\beta}_0\)
    3. \(\hat{\beta}_0 + \hat{\beta}_1x_{1i} + \hat{\beta}_2x_{2i} - \hat{u}_i\)
    4. \(y_i - \hat{\beta}_0 - \hat{\beta}_1x_{1i} - \hat{\beta}_2x_{2i}\)

  1. True or False? \(R^2 = 1 - \frac{SS_{residual}}{SS_{total}}\) will never decrease if additional independent variables are added to the estimated model.

    1. True
    2. False

  2. Imagine there is a third variable, \(x_3\), such that \(\beta_3\) (its associated coefficient) is equal to 0 (\(\beta_3 = 0\)) in the model above. If we include \(x_3\) in our regression model and estimate all coefficients (including \(\beta_3\)):

    1. \(\hat{\beta}_0,\hat{\beta}_1,\hat{\beta}_2\) will be biased estimators
    2. \(\hat{\beta}_0,\hat{\beta}_1,\hat{\beta}_2\) will be unbiased estimators
