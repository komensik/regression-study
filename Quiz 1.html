Quiz 1

\[ y_i = \beta_0 + \beta_1x_{1i} + \beta_2x_{2i} + u_i \]

  1. In the model above, \(\beta_0\) is:

    1. The predicted value of \(y_i\), holding \(x_1\) and \(x_2\) constant
    2. The x-intercept
    3. The sample mean of \(y_i\)
    4. The predicted value of \(y_i\) when \(x_1\) and \(x_2\) are both 0

  2. For a linear regression model, like the one above, we generally assume that:

    1. \(\text{E}(u_i) = 0\)
    2. The correlation of \(u_i\) with all predictors is 0
    3. \(x_{1}\) and \(x_{2}\) are uncorrelated
    4. Both a and b
    5. All of a, b, and c

  3. In the model above, \(\beta_1\) tells us:

    1. The predicted value of \(y_i\) when \(x_{2i} = 0\)
    2. The predicted value of \(y_i\) when \(x_{1i} = 1\)
    3. The change in the predicted value of \(y_i\) for a 1-unit change in \(x_{1i}\)
    4. None of the above

  4. In general, for a regression with one independent variable, if all the simple linear regression (SLR) assumptions are met (linear in parameters,random sampling, sample variation in explanatory variable, and zero conditional mean of error, \(\text{E}(u|x)=0\)), then:

    1. OLS estimators for \(\beta_0\) and \(\beta_1\) are biased but consistent
    2. OLS estimators for \(\beta_0\) and \(\beta_1\) are unbiased
    3. OLS estimators for \(\beta_0\) and \(\beta_1\) are biased and inconsistent
    4. None of the above
